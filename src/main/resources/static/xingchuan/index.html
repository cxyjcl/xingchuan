星川资本

武汉星川资本管理有限公司是经中国证券投资基金业协会核准成立集证券投资、股权投资和风险投资于一体的专业私募基金管理公司。星川资本与各大券商及金融机构拥有良好的合作关系及强大的客户群体；星川资本成立于2015年7月。私募基金管理人登记编号：P1023408

星川资本年度总结大会暨2016年终盛宴

2017年1月9日下午3点，公司召开了年度总结大会，对2016年的工作情况进行了全面总结。会上，每位员工对照年度工作目标和完成情况进行了述职，夏总、陈总、胡总等高管对每位员工的述职都给予了极具建设性的点评。董事长洪昌明代表公司进行发言，对员工的工作表示肯定并寄予厚望，总结了公司2016年度的各项情况并将经验和大家进行了分享，最后给大家送上了新年祝福。
